### **31. 스프라이트 셰이프 사용하기**

![](https://velog.velcdn.com/images/jihyun418/post/95d4e4b1-5f63-4038-a83a-9abbc8e85e03/image.png)

#### 스프라이트 모양 만들기
> 2D Object -> Sprite Shape -> Closed Shape

- Closed Shape : 벽이 둘러싸여 전체가 이어져 반복되는 형태
- Open Shape : 벽이 완벽하게 둘러싸지 못한 상태

![](https://velog.velcdn.com/images/jihyun418/post/0adb2d56-ac43-4475-aff8-1d7712af3fb4/image.png)

inspector 부분에 Sprite Shape Controller 라는 부분에 Sprite Shape profile 생성한 후 이름을 변경(Snow profile)하여 넣었다.
이후 Texture에 이미지를 넣어 색상을 채워주고 Sprites에도 이미지를 넣어 겉 테두리를 눈 모양으로 덮었다. 

![](https://velog.velcdn.com/images/jihyun418/post/66c3d452-a896-4dd1-9d4e-7b751c369d68/image.png)

Sprite Shape Controller에서 Edit Spline을 눌러 sprite를 조절할 수 있는데 이를 이용해 맵을 만들어주었다.

### 32.엣지 콜라이더 사용하기
![](https://velog.velcdn.com/images/jihyun418/post/5b50cd3a-f528-4cd1-9959-9f080ce81649/image.png)

엣지 콜라이더 추가
- Ass Component -> Edge Collider 2D 

공 추가
- 방법 1. Sprite에서 Circle 추가 후 Collider 2D, Rigidbody 2D 컴포넌트 추가
- 방법 2. Physics에서 Dynamic Sprite 추가
이후 Rigidbody에 중력 적용된지 확인

공이 굴러갈 때 살짝 붕 떠서 굴러가는 문제
- Collider 아래 Offset의 값을 낮추면 해결 가능하다. (강의와 다르게 x값 y값을 조절하는데 여기서 y값을 조절하여 맞췄다)

맵의 눈이 쌓인 부분 조절
- Edit Spline을 전부 드래그해서 선택한 후 Height 값을 조절하면 가능하다.
- Sprite Editor을 눌러서 사용할 이미지 영역을 설정할 수 있다.

### 33. 시네머신 카메라 팔로우 

- 시네머신 : 여러 대의 카메라를 동작할 수 있게 해주는 패키지, 여러 카메라에 적용되는 규칙을 쉽게 만들 수 있다

- 메인카메라 : 플레이 화면

- 시네머신 브레인 : 메인 카메라의 행동에 대한 로직을 정의, 여러 대의 가상 카메라(빠르게 , 느리게 갈 때, 클로즈업 등)도 지정 가능 



> 
#### 설치 경로 
Window -> Package Manager -> Unity Registry -> Cinemachine 을 Install 클릭하여 설치

> 
#### 가상 카메라 생성 방법
Component 클릭 혹은 Hierachy 영역에서 우클릭-> Cinemachine -> Virtual Camera 생성

- Body -> Framing Transposer: 게임 내 특정 대상을 프로그래밍하여 가상 카메라의 바디를 이동하는 것
- Follow -> 카메라가 따라다닐 대상 지정


Screan X 값을 0.25로 조정하여 카메라 중앙 초점을 앞쪽이 보이도록 설정하였다.

### 34. 캐릭터 설정

![](https://velog.velcdn.com/images/jihyun418/post/b16fd7a7-ef37-447d-8ade-f88b081fee1d/image.png)


캐릭터의 상체와 하체를 이전까지 공으로 설정했던 공 파일의 자식으로 넣었다. 
그렇게 캐릭터의 상하체 위치를 조절한 후 capsula Collider을 추가하였다. Direction을 Horizontal으로 변경한 후 size와 offset을 조절하여 보드 크기에 맞춰주었다. 
또한 넘어졌을 때 머리 부딪힘도 감지하기 위해 머리에도 Collider을 추가해주었다.  머리 Collider의 경우 Is Trigger를 체크하였다.
